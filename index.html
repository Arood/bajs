This is a JavaScript library for creating and/or managing CSS animations from your JS code.

Easy to use

This API was designed to be as simple as possible so you can spend your time making animations instead.

Gotta go fast

By utilizing CSS3 we guarantee optimal performance with hardware accelerated animations.

No dependencies

Use this library without any additional frameworks. It also handles vendor prefixes so you don't have to.

Getting started

Download the ba.js file from GitHub or link it externally:

https://cdn.rawgit.com/Arood/bajs/master/ba.js

Then check out the API reference and examples below.

Reference

Animate: .a(element, options, [callback])

element
A single HTML element that ba.js will keep an eye on.
options
An object containing options for the animation. See below for a list of options that can be used, additional keys will be used as keyframes in the animation. A keyframe consist of an object where every key-value is a CSS property. Transforms in a keyframe will automatically be vendor prefixed if needed.
callback (optional)
A function that is run after the animation has ended.

Options

curve
Specifies how a CSS animation should progress over the duration of each cycle. Supports any value that animation-timing-function can use.
delay
Specifies when the animation should start.
duration
Specifies the length of time that an animation should take to complete one cycle.
repeat
Defines the number of times an animation cycle should be played before stopping. Can be a number or "infinite", default is 1.

Animation Did End: .ade(element, callback)

element
A single HTML element that ba.js will keep an eye on.
callback
A function that is run after the animation has ended.

Availability Test: .at()

This method returns true if the animation property is supported in the current browser.
